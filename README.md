A web application for tracking fishing locations and experiences using an interactive map.

## Features

- Interactive map with markers for fishing locations
- Add, edit, and view fishing entries
- Track species, quantity, and notes
- View statistics and history
- Export/import entries

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm build
```

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES Modules)
- Leaflet.js (Map API)
- Open-Meteo Weather API
- Vite (Build Tool)
